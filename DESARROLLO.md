# Desarrollo del Proyecto

El proyecto se desarrolló en tres etapas principales:

1. **Análisis del problema:**  
   Se identificaron las principales limitaciones en el acceso a recursos educativos en zonas rurales.

2. **Diseño de la solución:**  
   Se propuso una aplicación web multiplataforma que permite compartir contenido educativo en formato abierto.

3. **Implementación:**  
   El desarrollo se realizó con tecnologías como:
   - HTML, CSS y JavaScript
   - Python (Flask)
   - Base de datos MySQL
